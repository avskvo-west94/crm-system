# Как внести вклад в проект

## Открытие иссю

Если вы нашли баг, пожалуйста, создайте Issue в репозитории.

## Pull Requests

1. Fork репозиторий
2. Создайте ветку для вашего фича
3. Коммитьте изменения
4. Push в открытые ветки
5. Открыть Pull Request

## Код стайл

- Отступ: 4 пробела
- Файлы Python: PEP 8
- Файлы TypeScript: ESLint

## Коммиты

От рекомендаций conventional commits:
- `feat:` новая фича
- `fix:` рип бага
- `docs:` изменения документации
